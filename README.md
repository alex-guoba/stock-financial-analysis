# stock financial analysis

## 功能描述

A股股票财报分析工具；用户索要个股财报、三大报表、股票财务分析报告、年报/季报拆解时自动启用；支持6位A股代码，可单独查利润表/资产负债表/现金流量表/全报表，支持年报year、季报quarter两种周期。

## 使用方式

### 直接使用

``` shell
cd scripts && poetry run python main.py {type} {code} {period}"
```

参数说明：
|参数|必填|可选值|说明|
|----|----|------|----|
|type|可选|all/profit/balance/cashflow|all=三大表全量；profit利润表；balance资产负债；cashflow现金流，默认all|
|code|必选|6位数字|A股股票代码，如600690、002XXX、300XXX|
|period|可选|year/quarter|year年报、quarter季报，默认year|


### 作为claude skill使用

1. 安装python+poetry环境
2. 复制文件到skills/stock-financial-analysis目录下
3. 运行`cd scripts && poetry install`
4. 启动claude，输入提示词

更多详细用法，参考 [SKILL.md](./SKILL.md)
